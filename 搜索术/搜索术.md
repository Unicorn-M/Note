# 搜索术

## 	搜索一般分为4个大类

1. 信息咨询

   一般是些新闻和大事件

2. 知识技能

   我们要学习一门技术,语言之类的教程

   还有一些概率文章，知识点等

3. 素材文件

   视频音频文件，还有一些文稿等素材文件

4. 工具软件

   在线工具、插件、软件

   ## 谷歌语法

   ```
   ""  限定关键词
   intitle  限定标题
   allintitle 限定标题多个关键字
   intext  限定内容关键词
   inurl   限定网址关键词
   site    限定网址来源
   imagesize   限定图片尺寸
   filetype  限定文件类型
   ```

    <h4 style="color:gray" >使用案例</h4>

   ```txt
   1.
   	直接搜索易阳千喜,会出来很多易烊千玺的信息
   	使用"易阳千喜"出来的就全是易阳千喜的信息
   	
   2.
   	intile:iphone15摄像头进灰   //出来的都是标题里有iphone15摄像头进灰的关键字(顺序不一定)
   
   	intile:"iphone15摄像头进灰" //出来的就是标题里有这几个关键字的，且顺序也不变
   
   
   3.
   	//如果标题里有多个关键词
   	allintile:李子柒回归 微念  //标题里有这两个的关键词的信息
   	
   4.	
   	//限定文件
   	大语言模型 filetype:pdf   //出来的就是很多大语言模型的研究报告
   
   5.
   	//查花西子眉笔成分信息
   	利用网站：国家普通化妆品备案信息
   ```
